# Data Engineering Zoomcamp 2026

Personal repository for the **DataTalksClub Data Engineering Zoomcamp 2026**.  
This repository contains my notes, homework assignments, exercises, and project work completed throughout the course.

## Course Overview

The Data Engineering Zoomcamp covers the fundamentals and practical aspects of modern data engineering, including:

- Data ingestion and pipelines
- Workflow orchestration
- Data warehousing and analytics engineering
- Batch and streaming processing
- Cloud infrastructure and best practices
